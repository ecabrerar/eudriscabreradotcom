Sistemas de Control de Versiones

La gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración se conoce como control de versión.

El control de versiones es útil para guardar cualquier documento que cambie con frecuencia, como una novela, libro o el código fuente de un programa.

Un sistema de control de versiones es software que administra el acceso a un conjunto de archivos, y mantiene un historial de cambios realizados.

En esta presentación vamos a explorar cómo funcionan los sistemas de control de versiones, su clasificación y cuales son los más utilizados.
