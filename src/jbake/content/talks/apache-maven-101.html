Primeros pasos con Apache Maven

Maven es una herramienta de software para la gestión y construcción de proyectos Java creada por Jason van Zyl, de Sonatype, en 2002.

Provee un enfoque integral para la gestión de proyectos de software. Desde la compilación, distribución, documentación y colaboración en equipo, Maven proporciona las abstracciones necesarias que fomenten la reutilización y toman gran parte del trabajo de generaciones de proyecto.

En esta charla vamos a desglozar los pasos básicos para empezar a crear proyectos usando Apache Maven. Así, como también, vamos a explorar algunas opciones que utilizamos en el día a día.
