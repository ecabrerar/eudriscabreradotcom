Simplifica el desarrollo de tus aplicaciones empresariales con Jakarta EE

Jakarta EE, también conocida como Java EE, es una plataforma estable y ampliamente adoptada para aplicaciones empresariales. Sin embargo, tiene fama de vieja y anticuada, ¡injustamente! Con las versiones e implementaciones actuales, la plataforma está ganando impulso. Jakarta EE contiene nuevas características para crear aplicaciones Java modernas, simplificadas, lijeras y orientadas a la nube.

Jakarta EE, ahora impulsada por la comunidad, ha introducido mejoras que son de gran ayuda para las organizaciones que desean iniciar el desarrollo de nuevas aplicaciones Java empresariales o mejorar las que ya tienen.

En esta sesión, demostraré cómo comenzar a desarrollar nuevas aplicaciones Java empresariales o cómo migrarlas rápidamente de Java EE 8 a Jakarta EE 10 o posteriores y ejecutarlas en entornos de ejecución ligeros como WildFly Bootable Jar, OpenLiberty, Payara o Quarkus.
