El 23 de mayo 1995 fue un día trascendental en la historia de la tecnología. La industria ignoraba  que el lanzamiento de un lenguaje de programación iba a cambiar por completo el panorama de la tecnología en los próximos años.

![](/images/blog/2015/java-20yr-comm-mark-horz-clr-300x121.jpeg)

## Qué es Java ?.
Es un lenguaje de programación de propósito general, concurrente, orientado a objetos que fue diseñado específicamente para tener tan pocas dependencias de implementación como fuera posible.

![](/images/blog/2015/James_Gosling_2008-298x300.jpg)
James Gosling, Creador de Java

El lenguaje de programación Java fue originalmente desarrollado por James Gosling de Sun Microsystems (la cual fue adquirida por la compañía Oracle).

Las aplicaciones de Java son generalmente compiladas a bytecode (clase Java) que puede ejecutarse en cualquier máquina virtual Java (JVM) sin importar la arquitectura de la computadora subyacente.

Si retrocedemos 20 años en el carril de la memoria y pensar en cuándo / cómo se inventó Java. World Wide Web era para las élites, 5 1/2 “disquetes, módems de acceso telefónico, y  el comercio electrónico se acababa de lanzar.

No existían los teléfonos inteligentes, Netflix, Twitter, Facebook, o cualquier tecnología. Todas estas tecnologías están ahora alimentado por Java!. Lo que demuestra cómo ha evolucionado el lenguaje y la plataforma a través de todos estos años.

El lenguaje C fue inventado ~ hace 43 años, C++ ~ hace 32 años, Objective C ~ hace 32 años, y C# ~ hace 15 años.
Java es el único lenguaje de programación distinto de C que se ha mantenido entre los 2 principales del Indice Tiobe.
